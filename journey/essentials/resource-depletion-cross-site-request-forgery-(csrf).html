{"Resource Depletion":"Cross-Site Request Forgery (CSRF)"}

An attack that forces an end user to execute unwanted actions on a web application in which theyre currently authenticated, targeting state-changing requests, not theft of data, since the attacker has no way to see the response to the forged request.
