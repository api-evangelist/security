Versign OpenHybridTM is a Verisign DDoS Protection Services architecture that enables interoperability between on-premise and cloud-based platforms.An easy to use tool (Open API), allows customers to leverage existing security perimeter devices to signal threat information to the Verisign DDoS protection cloud, when the capabilities of the device are exceeded.Verisign OpenHybrid can also be applied to monitor and protect services hosted within other public and private cloud services.Verisigns API helps create an interoperable and open ecosystem between multiple devices and cloud platforms, which Ensures application availability across data centers, public and private clouds, in the face of evolving DDoS attacks.Provides increased visibility of threats across diverse environments (both on premises and in public/private clouds).
